## 🤾‍♂️ 트러블슈팅
* 문제1<br>
 카카오톡 access token 값에서 id 추출 안되는 문제<br>
 access token 출력결과 길이가 길지 않아서 string으로 받았으나 long 자료형으로 받아 문제해결
 
* 문제2<br>
  그룹방마다 다른 그룹인덱스 값을 받아오지 못해 채팅메세지 DB가 연결 안되는 문제<br>
  이미 페이지에 나온 그룹 이름을 PK로 바꾸고 그룹 이름 FK로 받아 채팅 메세지에 연결
